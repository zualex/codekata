# Formatted time

## Purpose
Implement function `getTime` which takes a number of minutes, and return to a time line.

## Example
```javascript
getTime(60); // 01:00
getTime(5); // 00:05
getTime(15); // 00:15
getTime(175); // 02:55
getTime(67); // 01:07
```
