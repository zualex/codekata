# Formatted time

## Purpose
Implement function `geTime` which takes a number of minutes, and return to a time line.

## Example
```javascript
geTime(60); // 01:00
geTime(5); // 00:05
geTime(15); // 00:15
geTime(175); // 02:55
geTime(67); // 01:07
```
